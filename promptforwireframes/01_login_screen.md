# Screen 1: Login / Registration — Google Stitch Prompt

## Prompt:

Design a high-fidelity mobile login screen for a premium banking app called **"NexusVault"**. Dark theme, modern fintech aesthetic. This is the first screen users see — it must immediately communicate security and trust.

### Screen Layout (Top to Bottom):

**Background:**
- Deep dark navy (#0F172A) base
- Subtle radial gradient glow of blue (#3B82F6 at 10% opacity) emanating from behind the logo area
- Very subtle geometric grid pattern or circuit-board lines in the background at 3-5% opacity (futuristic touch)

**Top Section (Logo & Branding):**
- Center-aligned NexusVault logo: A shield icon with an "N" integrated into it, rendered in a blue-to-indigo gradient (#3B82F6 → #6366F1)
- Below logo: App name "NexusVault" in 28px Extra Bold white text
- Below name: Tagline "Your Finances. Fortified." in 14px regular #94A3B8 text
- Generous top padding (80px from status bar)

**Middle Section (Login Form):**
- "Welcome Back" title in 22px Bold white, left-aligned
- "Sign in to your account" subtitle in 14px #94A3B8, left-aligned
- 24px gap

- **Email/Phone Field:**
  - Dark card background (#1E293B) with 1px border (#334155)
  - Left icon: 📧 mail icon in #64748B
  - Placeholder text: "Email or phone number" in #64748B
  - Rounded corners (12px)
  - Height: 52px

- 12px gap

- **Password Field:**
  - Same styling as email field
  - Left icon: 🔒 lock icon in #64748B
  - Placeholder: "Password"
  - Right icon: 👁️ eye icon (show/hide toggle) in #64748B
  - Rounded corners (12px)
  - Height: 52px

- 8px gap

- **"Forgot Password?"** — Right-aligned, 13px, #3B82F6 (accent blue), tappable link

- 24px gap

- **"Sign In" Button:**
  - Full width
  - Height: 52px
  - Background: Linear gradient (#3B82F6 → #6366F1)
  - Text: "Sign In" in 16px Semibold white, centered
  - Rounded corners (12px)
  - Subtle blue glow shadow beneath the button (0 4px 14px rgba(59,130,246,0.4))

- 16px gap

- **Divider:**
  - Thin line (#334155) with "OR" text centered in #64748B on a small dark background pill

- 16px gap

- **"Sign in with Biometrics" Button:**
  - Full width
  - Height: 52px
  - Background: Transparent with 1px border (#334155)
  - Left icon: Fingerprint icon in #3B82F6
  - Text: "Sign in with Biometrics" in 15px Medium white
  - Rounded corners (12px)

**Bottom Section:**
- 32px gap
- Center text: "Don't have an account? " in #94A3B8 + **"Register"** in #3B82F6 (tappable)
- 24px gap
- **Security Badge:** A small pill/tag centered at the bottom:
  - Background: #1E293B with 1px border (#334155)
  - Content: 🔒 icon + "Secured by TLS 1.3 · AES-256 Encrypted" in 11px #64748B
  - Rounded corners (20px / full pill shape)
  - Subtle padding (8px 16px)

**NO bottom navigation bar on this screen** (user is not logged in yet).

### Design Keywords:
Dark mode, fintech, glassmorphism subtle, premium banking, trust, security-focused, clean, minimal, professional, futuristic
